# Typography ✍️

## Learning Objectives

- Style text with Tailwind utilities
- Use font families, sizes, and weights
- Control line height and letter spacing
- Apply text alignment and decoration

## Font Families

```html
<p class="font-sans">Sans-serif font (default)</p>
<p class="font-serif">Serif font</p>
<p class="font-mono">Monospace font</p>
```

## Typography Patterns

### Heading Hierarchy

```html
<h1 class="text-4xl font-bold text-gray-900 mb-4">Main Heading</h1>
<h2 class="text-3xl font-semibold text-gray-800 mb-3">Subheading</h2>
<h3 class="text-2xl font-medium text-gray-700 mb-2">Section Heading</h3>
<p class="text-base text-gray-600 leading-relaxed">Body text paragraph.</p>
```

### Article Text

```html
<article class="max-w-2xl mx-auto">
  <h1 class="text-4xl font-bold text-gray-900 mb-4">Article Title</h1>
  <p class="text-gray-500 mb-6">Published on Jan 15, 2024</p>
  <p class="text-lg text-gray-700 leading-relaxed mb-4">
    This is the article introduction with larger text...
  </p>
  <p class="text-base text-gray-600 leading-relaxed">
    Regular article body text continues here...
  </p>
</article>
```

## Key Takeaways

✅ **Font size scale** - From xs to 9xl
✅ **Font weights** - thin (100) to extrabold (900)
✅ **Line height** - Control spacing between lines
✅ **Text colors** - Use gray shades for hierarchy
