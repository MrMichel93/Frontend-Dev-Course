## Overview

This directory contains hands-on exercises for practicing Tailwind CSS. Each exercise builds a real-world component using Tailwind's utility classes.

## Exercise Files

Each exercise is in a separate JSX file. Open each file and follow the instructions in the comments to build the component.

1. **exercise-1-profile-card.jsx** - Build a Profile Card
2. **exercise-2-navigation-bar.jsx** - Create a Navigation Bar
3. **exercise-3-pricing-table.jsx** - Design a Pricing Table
4. **exercise-4-contact-form.jsx** - Build a Contact Form
5. **exercise-5-blog-card-grid.jsx** - Create a Blog Card Grid
6. **exercise-6-alert-component.jsx** - Design Alert Components
7. **exercise-7-dashboard-sidebar.jsx** - Build a Dashboard Sidebar
8. **bonus-landing-page.jsx** - Complete Landing Page (Challenge!)

## How to Use

1. Open each exercise file in your code editor
2. Read the requirements carefully
3. Use Tailwind CSS utility classes to build the component
4. Test your components by integrating them into a React app
5. Reference the [Tailwind CSS Documentation](https://tailwindcss.com/docs) as needed

## Tips

- Start mobile-first - use `sm:`, `md:`, `lg:` prefixes for responsive design
- Use consistent spacing with Tailwind's spacing scale
- Test on different screen sizes
- Experiment with different utilities
- Focus on semantic HTML structure first, then add styling

## Solutions

Try to complete exercises before looking at solutions. You can find reference implementations in the `examples/` directory of this module (if available).
