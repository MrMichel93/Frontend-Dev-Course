## Overview

This directory contains comprehensive exercises combining all concepts from Module 01. Complete these to solidify your understanding before moving to Module 02.

## Exercise Files

Each exercise is in a separate JSX file. Open each file and follow the instructions in the comments to complete the exercises.

1. **exercise-1-arrow-functions.jsx** - Arrow Functions & Destructuring
2. **exercise-2-array-methods.jsx** - Array Methods (map, filter, reduce, etc.)
3. **exercise-3-async-javascript.jsx** - Async/Await and Promises
4. **exercise-4-combined-concepts.jsx** - Data Dashboard, Components, and Form Validation
5. **exercise-5-real-world-scenarios.jsx** - API Data Processing and Complex Data
6. **bonus-challenges.jsx** - Advanced Patterns (pipe, debounce, memoize)

## How to Use

1. Open each exercise file in your code editor
2. Read the comments carefully - they explain what you need to implement
3. Write your code in the designated areas
4. Test your solutions by running the files with Node.js or in a browser console
5. Try to complete exercises without looking at answers first!

## Answer Key

Answers are provided in separate file: `exercises-answers.md` (if available)

## Self-Assessment

After completing exercises, you should feel comfortable with:
- [ ] Writing arrow functions naturally
- [ ] Destructuring objects and arrays
- [ ] Using spread operator for immutable updates
- [ ] Transforming data with map/filter/reduce
- [ ] Handling async operations with async/await
- [ ] Chaining array methods
- [ ] Combining all concepts in complex scenarios
