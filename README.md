## ğŸ› ï¸ Setup Instructions

### 1. Install Required Software

**Node.js (v18 or higher)**
```bash
# Check if Node.js is installed
node --version

# If not installed, download from: https://nodejs.org/
```

**npm (comes with Node.js)**
```bash
# Check npm version
npm --version
```

**Git**
```bash
# Check if Git is installed
git --version

# If not installed, download from: https://git-scm.com/
```

### 2. Clone This Repository

```bash
git clone https://github.com/MrMichel93/Frontend-Dev-Course.git
cd Frontend-Dev-Course
```

### 3. Install VS Code (Recommended)

Download from: https://code.visualstudio.com/

### 4. Install VS Code Extensions

Essential extensions for this course:

- **ES7+ React/Redux/React-Native snippets** - JavaScript/React snippets
- **Tailwind CSS IntelliSense** - Tailwind class autocomplete
- **Prettier - Code formatter** - Consistent code formatting
- **ESLint** - JavaScript linting
- **Auto Rename Tag** - Rename HTML/JSX tags automatically
- **Bracket Pair Colorizer** - Matching bracket colors

Install them from VS Code Extensions marketplace (Ctrl/Cmd + Shift + X)

### 5. Verify Your Setup

Navigate to any example project and run:

```bash
cd 02-Tailwind-CSS-Fundamentals/examples/landing-page
npm install
npm run dev
```

If you see a local development server running, you're all set! ğŸ‰
