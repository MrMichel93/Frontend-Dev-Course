## Exercise 1: Profile Card Component

Create a ProfileCard component that displays:
- User avatar (image)
- User name
- User bio
- Social media links
- Follow button

**Requirements:**

- Apply styling with className

## Exercise 2: Todo List Application

Build a complete todo list with:
- Input to add new todos
- List of todos
- Mark todos as complete
- Delete todos
- Filter (All, Active, Completed)

**Requirements:**
- Use useState for state management
- Handle form submission
- Use unique keys for list items
- Show/hide based on filter

## Exercise 3: Counter with Multiple Operations

Create a counter that supports:
- Increment by 1
- Decrement by 1
- Increment by 5
- Reset to 0
- Cannot go below 0

**Requirements:**
- Use useState
- Multiple event handlers
- Conditional logic for minimum value

## Exercise 4: Form with Validation

Build a registration form with:
- Name field (required, min 3 characters)
- Email field (required, valid email format)
- Password field (required, min 8 characters)
- Submit button
- Display validation errors

**Requirements:**
- Controlled components
- Form validation
- Conditional error messages
- Prevent submission if invalid

## Tips for Exercises

1. **Start Simple** - Build basic structure first
2. **Add State** - Identify what data changes
3. **Handle Events** - Add interactivity
4. **Style Last** - Focus on functionality first
5. **Test** - Try different scenarios
6. **Refactor** - Extract reusable components

## Solution Approach

For each exercise:
1. Plan component structure
2. Identify props needed
3. Determine state requirements
4. Implement UI first
5. Add event handlers
6. Test functionality
7. Add styling
8. Refactor if needed
